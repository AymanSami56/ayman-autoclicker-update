# Ayman Auto Clicker - Update Server

المستودع المخصص لتحديثات برنامج **Ayman Auto Clicker**.

يحتوي على الملفات التالية:

- **update.zip**  
  يحتوي النسخة التنفيذية الجديدة للبرنامج. يتم تنزيله تلقائياً عند التحديث الصامت.

- **version.json**  
  يحتوي معلومات آخر إصدار متاح، ويُستخدم من قِبل البرنامج لتحديد ما إذا كان هناك تحديث جديد.

- **changelog.txt**  
  سجل التغييرات للإصدارات الجديدة.

## طريقة عمل التحديث

يقوم برنامج Ayman Auto Clicker بالخطوات التالية:

1. قراءة `version.json`
2. مقارنة رقم الإصدار الحالي مع latest_version
3. إذا كان هناك إصدار جديد:
   - يُظهر نافذة تحديث
   - يقوم بتنزيل update.zip
   - فك الضغط في مجلد البرنامج
   - إعادة تشغيل التطبيق

## إعداد التحديث الجديد

لتجهيز تحديث جديد:

1. عدل ملف `version.json` وحدّث:
   - latest_version
   - update_zip_url
   - download_url (اختياري)
   - notes

2. ضع ملف `update.zip` الجديد في المستودع.

3. حدّث changelog.txt بإضافة تفاصيل الإصدار الجديد.

4. قم بعمل commit و push.
